The project  **WildFlux: a sound experience, from intuition to reality** aims to create a collaborative space where curiosity, creativity, and playfulness come together with science, art, and sharing knowledge.

At the heart of the project is a simple idea: turning any stream of data into music, and using this unique language as a starting point to explore and share scientific topics.

**Got to the project website to know more : https://wildflux-experience.github.io/**
