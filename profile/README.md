The project  **WildFlux: a sound experience, from intuition to reality** aims to create a collaborative space where curiosity, creativity, and playfulness come together with science, art, and sharing knowledge.

At the heart is a simple idea: turning any stream of data into music, and using this unique language as a starting point to explore and share scientific interests.

**Visit the page https://wildflux-experience.github.io/ to know more !**

## The Thereminuscule

⚡ On GitHub : https://github.com/wildflux-experience/thereminuscule-engine

The *Thereminuscule*, or more precisely the `thereminuscule-engine`, is a Python program at the core of this project.
It is designed to automatically generate coherent melodic lines as MIDI tracks, while allowing any incoming data stream to interact with the music in real time.
Both on the notes themself and the expressive way they are played.

Check the [Thereminuscule Signal Library](https://wildflux-experience.github.io/thereminuscule-signal-library/) for a collection of signal comming from scientific experimentations in the real world that are ready to be played ! 

## The Nano-Modules

⚡ On GitHub : https://github.com/wildflux-experience/thereminuscule-engine

The *Nano-Modules* are simple, low-cost electronic devices typically based on Arduino boards that can turn physical quantities, in a broad sense, into digital data streams compatible with Thereminuscule. They act as a bridge between the real world and the virtual world. There are many ways to measure the effect of an action, a movement, or a state using a physical quantity and a suitable sensor, opening the door to a wide variety of Nano-Modules to imagine and create.
