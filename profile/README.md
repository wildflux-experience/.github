# WildFlux: a sound experience, from intuition to reality.

This project aims to create a collaborative space where curiosity, creativity, and playfulness come together with science, art, and sharing knowledge.

At the heart of the project is a simple idea: turning any stream of data into music, and using this unique language as a starting point to explore and share scientific topics.

# WildFlux : une expérience sonore, de l’intuition jusqu’au réel.

Ce projet ambitionne de créer un espace collaboratif où curiosité, création et amusement riment avec science, art et transmission.

Au cœur de cette démarche, une idée simple : transformer n’importe quel flux de données en musique, et utiliser ce langage singulier comme point de départ pour explorer et transmettre autour de thématiques scientifiques.
